*   **Segurança:** Este é um exemplo básico. Em um ambiente de produção, você deve usar senhas com hash (como `password_hash` e `password_verify`) e prepared statements para evitar SQL Injection.
*   **Banco de Dados:** Adapte os dados de conexão com o banco de dados (servername, username, password, dbname) para as suas configurações.
*   **Sessões:** As sessões são usadas para manter o usuário logado. Certifique-se de iniciar a sessão no início do script PHP.
*   **Redirecionamento:** Após o login bem-sucedido, o usuário é redirecionado para uma página protegida (pagina\_protegida.php). Você precisará criar essa página e verificar se o usuário está logado antes de permitir o acesso.
